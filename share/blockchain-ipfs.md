# IPFS

## 什么是 IPFS
IPFS 英文全称 InterPlanetary File System（星际文件系统），他是一个点对点的去中心化的分布式文件系统，通过 IPFS 协议，你可以让存储在 IPFS 系统上的文件，在全世界任何一个地方快速获取，且不受防火墙的影响。

## IPFS 的工作原理

### IPFS 对象
IPFS 对象是用来描述存储内容的，结构如下：

* Data
* Link
    * Name
    * Hash
    * Size

```
{
    "Data": "",
    "Links": [{
        "Name": "",
        "Hash": "",
        "Size": 262158
    }, {
        "Name": "",
        "Hash": "",
        "Size": 81
    }]
}
```
